## Installation

Add `wechat_pay` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:wechat_pay, "~> 0.8.0"},
    {:jason, "~> 1.0"} # or {:poison, "~> 4.0"}
  ]
end
```
