## Installation

Add `wechat_pay` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:wechat_pay, "~> 0.4.1"},
    {:jason, "~> 1.0"} # or {:poison, "~> 3.0"}
  ]
end
```
